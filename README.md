## Basic Command:

**Running Anaconda**
```python
source ~<computer-name>/anaconda3/bin/activate
```
**Open Anaconda**
```python
anaconda-navigator
```
**Deactivate Anaconda**
```python
conda deactivate
```
**Create Virtual Environtment**
```python
virtualenv <nama_folder>
```
**Running Virtual Environtment**
```python
source <nama_folder>/bin/activate
```
**Deactivate Virtual Environtment**
```python
deactivate
```
